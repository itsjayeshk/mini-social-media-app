# 📝 Mini Social Media App

A full-stack web application built with Express and EJS that allows users to create, edit, delete, and like posts with dynamic updates.

## 🚀 Features

* Create, edit, and delete posts
* Like / Unlike system (no spam, toggle logic)
* Dynamic UI updates using fetch (no page reload)
* Clean modern UI

## 🛠 Tech Stack

* Node.js
* Express.js
* EJS
* CSS
* JavaScript (Fetch API)
